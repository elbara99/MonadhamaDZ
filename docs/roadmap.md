## 7. MVP Definition

The MVP (Phase 1 completion) includes:

- **20 data sources** integrated (ONS, Health, Education, Economy, Infrastructure)
- **Knowledge graph** with 15 entity types across 58 provinces
- **Scoring engine** for all 12 dimensions
- **Natural language search** in Arabic, French, English
- **Basic AI question answering** with citations
- **Province explorer** with detail pages
- **User authentication** with role-based access
- **5 pilot users** from 2 ministries

## 8. Key Dependencies

| Dependency | Risk | Mitigation |
|---|---|---|
| Data source availability | High | Placeholder data, parallel engagement |
| Government approvals | High | Early stakeholder engagement |
| LLM API access | Medium | Self-hosted fallback models |
| GPU availability | Medium | Cloud GPU pre-provisioned |
| Domain expertise | Medium | Government liaison team |
